# Cisco Packet Tracer Simulations

A collection of enterprise network architectures, routing labs, and IoT simulations built in **Cisco Packet Tracer**. 

This repository serves as a working sandbox to practice core networking fundamentals, subnetting, and secure device transport.

## Active Topologies

* **[SmartCity IoT Infrastructure](./SmartCity-IoT-Infrastructure)**
  
  * **The Scenario:** A simulated city grid where smart streetlights and environmental sensors are isolated on a secure VLAN and routed to a central Web/CMS server.
  * **Core Focus:** Inter-VLAN Routing (Router-on-a-Stick), edge DHCP provisioning, Layer 2 isolation, and DNS resolution.

*(Future simulations focusing on dynamic routing, VPNs, and ACL security will be logged here as they are developed).*
